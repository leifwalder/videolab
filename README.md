# videolab

deploy instructions:
<br />

`npm install`

`npm run start`

<br />
dev instructions:
<br />

`npm install`

`npm run some-dev`

<br />

key input:
<br />
Left, right to change selected video (disabled in fullscreen)
<br />
Enter to play, pause video
<br />
Up to enter fullscreen (down to exit fullscreen)
<br />
Down to exit fullscreen or toggle view
